# originals/

Drop your full-resolution photos here.

**These files are NOT committed to git** (they're ignored on purpose — raw photos
would bloat the repo history forever). They live here only so the converter can
read them.

To turn them into web-ready images for the site, run from the project root:

```
npm run images
```

That generates optimised WebP files in `assets/images/`, which *are* committed.

See the export specs in the project notes (3:4 portrait, 1200×1600, JPG out of Canva).
